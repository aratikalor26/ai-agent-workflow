# 🤖 AI Agent Workflow

Built an AI agent that understands user queries, retains context, and uses external knowledge to generate personalized responses.

---

## 🚀 Overview
This project demonstrates a workflow-based AI agent that goes beyond a basic chatbot by integrating memory and external tools to deliver context-aware responses.

---

## ⚙️ Architecture
User Input → AI Agent → Memory + External Tool → Response

---

## 🔧 Tech Stack
- LLM: Google Gemini  
- Memory: Simple Memory  
- Tool Integration: Wikipedia  

---

## 💡 Key Features
- 🧠 Context-aware responses using memory  
- 🌐 External knowledge integration (Wikipedia)  
- 🔄 Workflow-based agent design (not just prompt-response)
